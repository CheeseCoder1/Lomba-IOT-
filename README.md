# Lomba-IOT-

Cloud Informatika UKWMS | Otelo Donuts adalah file .zip yang berisi semua aset yang digunakan dalam proyek serta semua file .html, .css, dan .javascript.

Website Project for Lomba IOT by Cloud Informatika UKWMS | Daniel, Dustin, Edward

Inilah cara menggunakan situs web tersebut: 

- Di bagian atas situs web, terdapat bilah navigasi yang memungkinkan pengguna untuk menuju ke bagian tertentu dari situs web ketika Anda mengklik artikel yang diinginkan. 

- Tangkapan layar di bawah ini menunjukkan tombol navigasi dan lokasi yang diinginkan.

![2024_09_28_0yh_Kleki](https://github.com/user-attachments/assets/20f92d2d-ebfc-4a67-9783-b16aaec26526)

1. Jika diklik, itu akan membawa pengguna ke bagian atas situs web.

2. Jika diklik, itu akan membawa pengguna ke bagian atas situs web.

3. Jika diklik, itu akan membawa pengguna ke bagian "Our Donuts".

4. Jika diklik, itu akan membawa pengguna ke bagian "About Us".

5. Jika diklik, itu akan membawa pengguna ke bagian "Best Sellers".

6. Jika diklik, itu akan membawa pengguna ke bagian "Locations".

- Berpindah ke elemen lain di situs web, jika pengguna mengklik tombol "Select Donuts", pengguna akan dibawa ke bagian "Best Sellers". 

![2024_09_29_0h9_Kleki](https://github.com/user-attachments/assets/a2efb058-7667-4f89-9003-ad484a34597d)

- Bagian tersebut memiliki tombol, tetapi saat ini belum memiliki fungsi. Kami berencana untuk menambahkan fitur di mana pengguna dapat menambahkan produk yang mereka pilih ke dalam keranjang dan di sana mereka dapat melakukan pembayaran. 

![2024_09_29_0hj_Kleki](https://github.com/user-attachments/assets/d6c73db4-aca2-4f29-831e-cde88f6e1ad4)

- Kami beralih ke tombol "See More Donuts" dan saat ini tombol tersebut tidak memiliki fungsi lain selain mengarahkan pengguna ke bagian atas situs web. Namun, kami berencana untuk menerapkan fungsi di mana pengguna akan diarahkan ke halaman lain di mana kami menyediakan berbagai pilihan donat dan produk lain yang mereka jual.

![2024_09_29_0vl_Kleki](https://github.com/user-attachments/assets/74bfc7d3-a521-44b5-8b79-f7b677dfb7fd)

- Kembali ke bagian "Best Sellers", seperti yang telah kami sebutkan sebelumnya, tombol-tombol saat ini tidak memiliki fungsi. Namun, kami berencana untuk menerapkan fitur di mana pengguna dapat menambahkan produk yang dipilih ke dalam opsi keranjang, memungkinkan mereka untuk melakukan pembayaran.

![2024_09_29_0w8_Kleki](https://github.com/user-attachments/assets/6ffa8f77-9cad-4d4f-b1be-943f8ecebd03)

- Menggulir ke bawah di bawah bagian "Best Sellers" menunjukkan kepada pengguna bagian "Visit Us". Tombol ini akan membawa pengguna ke tautan Google Map dari toko tersebut. 

![2024_09_29_0wf_Kleki](https://github.com/user-attachments/assets/5121a872-59f5-4266-9c36-d7d0ab991416)

- Di bagian akhir situs web, kita sampai di footer. Seperti yang Anda lihat di sudut kanan bawah footer, ada ikon-ikon ini dari Instagram, WhatsApp, dan sebuah pohon.  Ikon-ikon tersebut juga terlihat di bagian atas situs web di bawah lokasi. Jika seorang pengguna mengklik ini, mereka akan diarahkan ke tautan yang sesuai.
